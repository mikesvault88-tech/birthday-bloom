# Getting Started 🚀

Welcome to **Birthday Bloom**! This guide will help you set up your own premium birthday surprise engine in minutes.

## 🛠 Prerequisites
Before you begin, ensure you have:
- **Node.js**: (Version 18 or higher)
- **Git**: Installed on your machine.
- **A Code Editor**: VS Code is recommended.

## 🏃 Quick Installation

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/naborajs/birthday-bloom.git
    cd birthday-bloom
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Start the Engine**:
    ```bash
    npm run dev
    ```
    Your site will be live at `http://localhost:5173`.
